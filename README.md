# MNN-Transformer-Based Approaches for Spanish Satirical News and AI-Generated Humor Detection

This project presents a hybrid approach to classify Spanish text as satirical news, real news, or AI-generated humor, as part of the **HAHA 2026 shared task**. Our solution combines the power of pre-trained Spanish language models with a Multi-Negation Network (MNN) to capture complex linguistic patterns while mitigating the effects of adversarial attacks.

## Overview

The **HAHA 2026 shared task** aims to detect three classes of Spanish text:
1.  **News**: Genuine news articles.
2.  **Satire**: Satirical news articles.
3.  **AI-Generated Humor**: Content humorously generated by large language models.

Our approach focuses on two main components: a state-of-the-art **Transformer-based classifier** and a **Multi-Negation Network (MNN)** module designed to handle negation and adversarial perturbations.

## Methodologies

We explored three main methodologies:

### 1. Multi-Task Learning (MTL) with 3-Head Classifier

This model is designed to capture the nuances of all three classes using a shared encoder. The task is modeled as a multi-label classification problem where each class is treated as a separate head.

**Model Architecture:**
- **Base Encoder**: A pre-trained Spanish language model (e.g., `BSC-CSL/RoBERTa-NE-Spanish`) is used to generate contextual embeddings.
- **Task Heads**: Three distinct feed-forward networks (FFNs) are connected to the encoder's output, one for each task:
    - **Task 1**: Distinguishing News from Non-News.
    - **Task 2**: Detecting Satire.
    - **Task 3**: Detecting AI-Generated Humor.
- **Classification**: The final output is a weighted sum of the probabilities from the three heads.

**Training Strategy:**
- **Data**: The dataset is augmented with additional Spanish satirical news from external sources to enhance generalization.
- **Multi-Task Loss**: The total loss is a weighted sum of the individual task losses.

### 2. Multi-Negation Network (MNN) with Dual Transformers

This approach specifically addresses the challenges of negation and adversarial attacks by using two parallel transformer encoders that process the text in different "flipped" states.

**Model Architecture:**
- **Dual Encoders**: Two identical transformer encoders operate in parallel:
    - **Encoder 1**: Processes the original text.
    - **Encoder 2**: Processes the text with all negation tokens inverted (e.g., "no" -> "si").
- **MNN Layer**: A specialized layer that combines the outputs of the two encoders.
    - It uses element-wise multiplication to "fuse" the representations, effectively learning to identify patterns that are invariant to or amplified by negation.
    - This structure makes the model robust to adversarial perturbations that rely on minor changes in negation.
- **Classifier**: A simple linear classifier on top of the fused representation.

### 3. RoBERTa-Based Classifier

A robust baseline model using the state-of-the-art RoBERTa architecture fine-tuned on the specific Spanish dataset.

## Key Innovations

1.  **Multi-Task Learning**: By framing the problem as three related tasks, the model learns a richer, more generalizable representation of Spanish text.
2.  **Negation Flipping**: The MNN approach explicitly handles negation by processing both original and inverted text, making it robust to adversarial manipulations.
3.  **Hybrid Architecture**: Combining the power of large language models with specialized network modules for specific linguistic challenges.

## Conclusion

Both the **Multi-Task Learning** and **Multi-Negation Network** approaches demonstrate strong performance on the HAHA 2026 shared task. The MTL model achieves the second-best result overall, highlighting the effectiveness of multi-task learning for complex classification problems. The MNN model provides valuable insights into handling negation and adversarial robustness in satirical content detection.
